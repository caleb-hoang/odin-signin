# odin-signin

A mock webpage simulating a sign-in page for an imaginary service, made as a project for the "Odin Project: Intermediate HTML" course. Features form validation, custom form stylization. Currently not responsive to mobile layouts.

To run locally: Open index.html in your browser of choice.
